# Agent OS - 分层自治系统

> **一句话，自动完成所有工作**
> 
> 你只需要说一句话，系统自动：
> - 识别你想做什么（金融分析？生成梗图？查电商优惠？）
> - 自动搜索相关信息
> - 自动分析决策
> - 自动推送结果或保存草稿

---

## 🚀 三步上手

### 第一步：安装依赖
```bash
pip install -r requirements.txt
```

### 第二步：配置API Key（可选）
```bash
# 复制配置文件
cp .env.example .env

# 用记事本打开 .env 文件，填写你有的API Key
```

| API Key | 用途 | 获取地址 |
|---------|------|----------|
| API_DEEPSEEK | 智能分析 | https://platform.deepseek.com/ |
| API_SERVERCHAN | 推送微信 | https://sct.ftqq.com/ |
| API_BING_SEARCH | 搜索热点 | https://azure.microsoft.com/ |

> **没有API Key也能跑！** 系统会自动使用内置模板和模拟数据。

### 第三步：开始使用
```python
# 方式一：运行演示
python demo.py

# 方式二：交互式对话
python main.py
```

## 🏗️ 分层架构

```
用户输入一句话 → 领域识别 → 自动执行

┌──────────────────────────────────────────────────────┐
│                   搜索层 (Search Layer)              │
│  自动决定去哪搜：金融→coingecko，搞笑→微博，电商→京东  │
├──────────────────────────────────────────────────────┤
│                   思考层 (Think Layer)               │
│  LLM自动分析，RAG做共享记忆，层次间自然沟通            │
├──────────────────────────────────────────────────────┤
│                   行动层 (Action Layer)              │
│  自动决定推送方式：微信、邮箱、保存草稿箱             │
└──────────────────────────────────────────────────────┘
```

## ⚙️ 配置说明

打开 `.env` 文件，只需填写你有的：

```
# DeepSeek大模型（智能分析用）
API_DEEPSEEK=你的Key

# ServerChan（推送微信用）
API_SERVERCHAN=你的Key

# 是否推送微信
PUSH_TO_WECHAT=true

# 自动发布间隔（分钟）
INTERVAL_MINUTES=30
```

## 📌 核心逻辑

1. **领域识别**：系统自动判断你想做什么
2. **搜索层**：根据领域自动选择数据源
3. **思考层**：用LLM分析，RAG记住上下文
4. **行动层**：自动推送或保存

> **无需懂代码，无需懂API，无需懂逻辑**
> 
> 你只需要说一句话，剩下的交给系统！
