# Agent OS - 分层自治系统

> 一句话，系统自动完成所有工作。无需懂代码，无需懂API，无需懂逻辑。

## 核心设计

```
用户输入一句话 → 领域识别 → 自动执行

┌──────────────────────────────────────────────────────┐
│                   搜索层 (Search Layer)              │
│  LLM自动决定去哪搜：金融→coingecko，搞笑→微博，电商→京东  │
├──────────────────────────────────────────────────────┤
│                   思考层 (Think Layer)               │
│  LLM自动分析，RAG做共享记忆，层次间自然沟通            │
├──────────────────────────────────────────────────────┤
│                   行动层 (Action Layer)              │
│  LLM自动决定推送方式：微信、邮箱、保存草稿箱             │
└──────────────────────────────────────────────────────┘
```

## 快速开始

```python
from agent_os.core.agent_os import agent_os

# 用户只需要说一句话，系统自动完成所有工作
agent_os.run("帮我分析今天的金融市场")
agent_os.run("给我生成一个搞笑梗图")
agent_os.run("查一下电商优惠信息")
```

## 已支持的领域

| 领域 | 数据源 | 自动执行 |
|------|--------|----------|
| 金融 | coingecko, yahoo finance, finviz | 搜索→布林带分析→LLM决策→微信推送 |
| 热点/搞笑 | 微博, 抖音, imgflip | 搜索→生成文案→保存草稿箱 |
| 电商 | 什么值得买, 京东, 淘宝 | 搜索→分析 |
| 科技 | 新浪科技, IT之家 | 搜索→分析 |
| 新闻 | 新浪新闻, 百度 | 搜索→分析 |
